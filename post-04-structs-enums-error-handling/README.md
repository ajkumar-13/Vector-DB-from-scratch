## Key Concepts

- **Structs**: Named fields, tuple structs, newtype pattern
- **impl blocks**: Methods and associated functions
- **Enums**: Algebraic data types with embedded data
- **Option<T>**: Rust's null-safe alternative
- **Result<T, E>**: Explicit error handling
- **? operator**: Error propagation shorthand
